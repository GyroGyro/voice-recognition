## Dependencies
This script require the follwing modules/libraries:
* [altgraph](https://pypi.org/project/altgraph/) 
* [macholib](https://pypi.org/project/macholib/) 
* [modulegraph](https://pypi.org/project/modulegraph/) 
* [py2app](https://pypi.org/project/py2app/) 
* [PyAudio](https://pypi.org/project/PyAudio/) 
* [pyobjc](https://pypi.org/project/pyobjc/) 
* [pyttsx](https://pypi.org/project/pyttsx/) 
* [SpeechRecognition](https://pypi.org/project/SpeechRecognition/) 
* [pocketsphinx](https://pypi.org/project/pocketsphinx/)

Libs can be installed as follows:
```
pip install -r requirements.txt
```
